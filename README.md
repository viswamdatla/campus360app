# campus360app

Monorepo containing:

- **Web app (React + Vite + SCSS)** at repo root
- **Mobile app (React Native + Expo)** at `expo-faculty/`

## Run the web app

```bash
npm install
npm run dev
```

## Run the Expo (React Native) app

```bash
cd expo-faculty
npm install
npm start
```

Then open it in **Expo Go** (Android) / emulator, or press `w` for web.
